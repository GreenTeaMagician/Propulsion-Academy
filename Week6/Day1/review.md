# JQuery review

Let's take a quick review at jQuery and `$` so far:

- `$(<CSS Selector as string)`
- `$(<HTML Tag as string>)`
- `$(<function>)`
- `$(HTML Element)`

JQuery object methods

- `.append(<JQuery Object or HTML as string>)`
- `.text(<string>)`
- `.html(<HTML tag as string or JQuery object)`
- `.remove()`, `.empty()`
- `.attr(<string>)`, `.attr(<string>, <string>)`
- `.on(<string>, <function>)`
