= Merge sort

"Successively merge sorted sets" 

* Merging two already sorted arrays is fast
* Divide and conquer
   - Make problem really small (DIVIDE)
   - Sort really small array (CONQUER)
   - Then recursively go back
   - And merge the already sorted arrays
